# HR Analytics Dashboard (Excel)

## About This Project
This project is an HR Analytics Dashboard created using Microsoft Excel. 

It focuses on analyzing employee data to understand important factors like employee retention, performance, salary distribution, and recruitment sources. The dashboard is designed to present data in a simple and clear way so that it is easy to understand and useful for decision making.

---

## Project Objective
The main objective of this project is to turn raw HR data into meaningful insights using Excel.

---

## Dataset Information
The dataset includes the following information:
- Employee ID
- Department
- Salary
- Gender
- Performance Score
- Absences
- Recruitment Source
- Employment Status
- Termination Reason

Note: The dataset is used only for learning purposes.

---

## Data Cleaning
The following steps were performed during data cleaning:
- Removed duplicate records
- Handled missing values
- Standardized text data
- Created calculated columns where needed

---

## Dashboard Features
The dashboard includes:
- Employee distribution by department
- Average salary analysis
- Recruitment source analysis
- Gender distribution
- Employee turnover analysis
- Performance vs attendance comparison

---

## Key Insights
- The production department has the highest number of employees
- Online recruitment is the most commonly used source
- Employee performance is affected by absences
- Some departments show higher turnover rates
